CSV Format Requirements

The CSV file must contain exactly these 5 columns in order, without any headers missing. The first row (headers) will be skipped.

national_id,first_name,last_name,email,academic_year
299051512345,أحمد,محمد,ahmed@example.com,1
299051512346,سارة,علي,sara@example.com,2
  • national_id: Used as both the Username and default Password.
  • academic_year: Integer (1=First Year, 2=Second Year, etc.).
  • section_group: Automatically calculated based on the first Arabic letter of the first name.
